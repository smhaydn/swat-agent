---
description: Systematic 4-phase debugging and root cause analysis
---

1. Phase 1: REPRODUCTION. Ask for logs, steps to reproduce, or environment details.
2. Phase 2: ISOLATION. Locate the faulty file or function.
3. Phase 3: FIXING. Apply the fix according to Coding.md standards.
4. Phase 4: VERIFICATION. Verify the fix and ensure no regressions.
